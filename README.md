#  AI Resume Analyzer

An AI-powered Resume Analyzer built using Python, Streamlit, and NLP techniques to evaluate resumes against job descriptions.


##  Features

- Resume PDF Upload
- Job Description Matching
- ATS Score Prediction
- Missing Skills Detection
- Skill Match Visualization
- Interactive Dashboard UI

---

##  Tech Stack

- Python
- Streamlit
- Scikit-learn
- NLP
- TF-IDF Vectorizer
- Cosine Similarity
- Matplotlib
- pdfplumber

---

##  Project Workflow

1. Upload Resume PDF
2. Paste Job Description
3. Analyze Resume
4. Generate:
   - Match Score
   - ATS Score
   - Missing Skills
   - Skill Analysis Chart

##  Installation

Clone the repository:

```bash
git clone https://github.com/HarshAwasth-i/Resume_Analyzer.git
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the project:

```bash
streamlit run app.py
```
