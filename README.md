 AI Resume Analyzer 

A full-stack web application that analyzes resumes using NLP and matches skills with job descriptions.

Features
- Upload resume (PDF)
- Extract skills using spaCy
- Match with job skills
- Show match score and missing skills

 Tech Stack
- Frontend: React
- Backend: Flask
- NLP: spaCy

 Run Locally

Backend
cd backend  
python app.py  

 Frontend
cd frontend  
npm start
